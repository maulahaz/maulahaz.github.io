Project Description

Handover is a Taken-and-Returned Inventory Tracking System — a full-stack Next.js application for managing tool handovers between shifts with condition reporting and acknowledgement. Each handover records the item used, the user who took it, the user who received it, start and end conditions, and acknowledgement status. The system is designed mobile-first and responsive for desktop use.

Project information

  • Category: Web Apps
  • Tech Stack: Next.js, MongoDB, Prisma, Tailwind CSS
  • Project date: 1st week of March 2026
  • Project URL: hand-over.netlify.app

Features :

  • Taken-and-Returned Inventory Tracking
  • Shift Handover Management
  • Item Condition Reporting
  • Acknowledgement Status Tracking
  • Mobile-first Responsive Design
  • Prisma ORM with MongoDB
